# User Prompt
On the left side menu, create a new menu item with label "Admin Mode".
On this screen, there should be a password field.
On entering the right password, that user becomes an 'Admin' user.
The right password is stored in the application properties file, stored somewhere in app/src/main/res/

# Admin Mode & Package Renaming Implementation Plan

This plan details the implementation of the Admin Mode feature alongside completing the package renaming from `com.example` to `com.vypeensoft`.

## User Review Required

> [!IMPORTANT]

> - The Java source files will have their package headers and resource imports updated to `com.vypeensoft.videochatapp`.
