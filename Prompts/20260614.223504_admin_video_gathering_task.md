# User Prompt
once in admin mode, when the video call starts, admin user sends the other users the information that he is the Admin user .
Once the Other user get this message, the others have to start sending the currently recorded video to him.  
Since the video are of 60 seconds each, the other users start sending those pieces one by one, starting with the earliest piece.
The admin user have to start saving the MP4 files for each of the other users in the video call.
Make sure that there is no conflict when saving files....
