# Implementation Plan - Audio Recording Support

Introduce audio recording capability to the WebRTC video recorder, enabling local `.mp4` calls to contain both audio and video, while keeping remote recordings video-only.

## User Review Required

> [!NOTE]
> Playout/remote audio cannot be captured via standard Java WebRTC APIs due to SDK limitations (lack of an `AudioTrack.addSink` equivalent). Remote recordings will remain video-only, while local recordings will capture local video + local microphone audio.

## Proposed Changes

### WebRTC Call and Recording Component

#### [MODIFY] [WebRtcVideoRecorder.java](file:///x:/Projects_X/0_Active/8_Android_APK/04_Early_Stage/Video_Chat_APK_GITHUB/app/src/main/java/com/example/videochatapp/webrtc/WebRtcVideoRecorder.java)
- Update constructor to accept `boolean recordAudio`.
- Implement dynamic initialization for `MediaCodec` (AAC audio encoder) upon receiving the first audio buffer.
- Add an asynchronous queue and thread (`AudioEncoderThread`) to drain audio samples and feed them to the encoder.
- Update `MediaMuxer` startup logic to coordinate both video and audio tracks:
  - If `recordAudio` is true, wait for both video and audio track indices to be resolved before calling `mediaMuxer.start()`.
  - If `recordAudio` is false, start the muxer as soon as the video track is resolved.

#### [MODIFY] [WebRtcClient.java](file:///x:/Projects_X/0_Active/8_Android_APK/04_Early_Stage/Video_Chat_APK_GITHUB/app/src/main/java/com/example/videochatapp/webrtc/WebRtcClient.java)
- Expose a method to register local audio recorders or pass raw microphone samples.
- Configure `JavaAudioDeviceModule` with `setSamplesReadyCallback` to intercept raw microphone PCM samples and forward them to the local recorder.

#### [MODIFY] [CallActivity.java](file:///x:/Projects_X/0_Active/8_Android_APK/04_Early_Stage/Video_Chat_APK_GITHUB/app/src/main/java/com/example/videochatapp/activities/CallActivity.java)
- Instantiate `localRecorder` with `recordAudio = true`.
- Instantiate `remoteRecorder` with `recordAudio = false`.
- Register the local recorder to receive microphone samples from `WebRtcClient`.
