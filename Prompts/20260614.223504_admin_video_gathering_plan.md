# Admin Video Gathering Implementation Plan

This plan details the implementation of a video transmission feature where an Admin user triggers non-admin users to send their recorded 60-second video segments sequentially over the signaling WebSocket.

## User Review Required

> [!IMPORTANT]
> - **Video Chunking**: Video files will be read in chunks (e.g. 64KB) and sent as Base64 encoded strings via the WebSocket signaling channel to avoid payload size limitations.
> - **File Savings**: Received files will be saved in the Admin's recording directory with filenames prefixed with `received_<senderId>_` to avoid conflicts.
