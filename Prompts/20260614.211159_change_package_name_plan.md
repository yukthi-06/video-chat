# Package Name Refactoring Plan

This plan details the steps required to change the application package name from `com.example` to `com.vypeensoft`. This affects both the Android namespace/applicationId and the Java package naming structure/directories.

## User Review Required

> [!IMPORTANT]
> The source directory structure will be moved from `app/src/main/java/com/example/videochatapp` to `app/src/main/java/com/vypeensoft/videochatapp`. This will change the absolute file paths of all the Java class files in this package.

## Proposed Changes

### Build Configuration & Manifests

#### [MODIFY] [build.gradle](file:///x:/Projects_X/0_Active/8_Android_APK/04_Early_Stage/Video_Chat_APK_GITHUB/app/build.gradle)
- Update `namespace 'com.example.videochatapp'` to `namespace 'com.vypeensoft.videochatapp'`.
- Update `applicationId "com.example.videochatapp"` to `applicationId "com.vypeensoft.videochatapp"`.

#### [MODIFY] [AndroidManifest.xml](file:///x:/Projects_X/0_Active/8_Android_APK/04_Early_Stage/Video_Chat_APK_GITHUB/app/src/main/AndroidManifest.xml)
- Update `package="com.example.videochatapp"` to `package="com.vypeensoft.videochatapp"`.

#### [DELETE] `app/src/main/java/com/example/videochatapp`
- Move all files from `com/example/videochatapp` to `com/vypeensoft/videochatapp`, then delete the `com/example` directory if it is empty.
